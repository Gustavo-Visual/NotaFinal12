Como funciona?

Calcular

Insere as tuas notas nas 3 componentes e a média final é calculada automaticamente. Cada componente vale ⅓ do resultado.

(📚 + 💼 + ⭐) ÷ 3 = Nota Final

Componentes EPROMAT

📚 Disciplinas — Média das disciplinas do curso
💼 Estágio — Formação em contexto de trabalho
⭐ PAP — Prova de Aptidão Profissional

Simular

Define um objetivo e preenche as notas que já sabes. O calculador mostra o mínimo que precisas nas restantes componentes. Usa os sliders para explorar cenários rapidamente.

Personalizado

Cria a tua própria estrutura de avaliação com componentes e pesos personalizados. Ideal para ensino regular, superior, ou qualquer sistema de notas.
